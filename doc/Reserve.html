#create_hand ⇒ Array<Array(Integer, String)>

Calls draw_card instance method six time to create a hand and deletes any nil elements that may arise from empty array.

Array#delete_if used over Array#delete due to the latter returning the deleted item rather than the array.

def create_hand
  hand = []
  6.times { hand << draw_card }
  return hand.delete_if { |card| card == nil }.sort
end

def draw_card
  card = @reserve.sample
  @reserve -= [card]
  return card
end
